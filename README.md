# Agenda Telefónica

## Descripción
Aplicación de consola en Java que permite gestionar una agenda de contactos donde cada contacto puede tener múltiples números de teléfono.

## Funcionalidades
- Añadir contactos nuevos
- Asignar múltiples teléfonos a cada contacto
- Evitar contactos duplicados
- Mostrar agenda completa con todos los teléfonos

## Tecnologías
- Java
- HashMap (contactos)
- HashSet (teléfonos por contacto)
- Scanner
